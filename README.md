
# Dictionary

App for saving entries and searching among them.




## Deploy

To Deploy this project

```bash
  ./gradlew clean build docker dockerComposeUp
```

## Tech Stack

- Java 17
- Spring Boot 2.6.5
- MongoDb 4.4.13
- Elasticsearch 7.16.3
- Kafka 3.2.0
